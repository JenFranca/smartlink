# 🌐 Smart Link Internet Fibra — Site Institucional

Site institucional desenvolvido para a **Smart Link Internet Fibra**, provedor de internet fibra óptica. Projeto focado em apresentar os planos, diferenciais da empresa e facilitar o contato com clientes.

🔗 **[Ver site ao vivo](https://jenfranca.github.io/smartlink)**

---

## ✨ Destaques do projeto

- Design **dark neon** moderno com tema tech
- Animações com **Canvas API** — partículas e fibras ópticas no fundo
- **4 planos** de internet com cards interativos
- Botões de contratação direto pelo **WhatsApp**
- Seção de contato com **WhatsApp**, **Telefone** e **Instagram**
- **100% responsivo** — mobile, tablet e desktop
- Sem frameworks — HTML, CSS e JavaScript puros

---

## 🛠️ Tecnologias utilizadas

- **HTML5**
- **CSS3** — variáveis, grid, flexbox, animações
- **JavaScript Vanilla** — Canvas API, IntersectionObserver, animações
- **Font Awesome 6.5.0** — ícones via CDN

## 📡 Seções do site

1. **Navbar** — navegação fixa com efeito scroll
2. **Hero** — logo, headline e stats animados
3. **Diferenciais** — 6 cards com ícones
4. **Planos** — 200mb / 400mb / 600mb / Mesh
5. **Frase de impacto** — destaque tipográfico
6. **Contato** — WhatsApp, Telefone e Instagram
7. **Footer** — 4 colunas com links e informações
